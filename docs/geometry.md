# BookCabinet — Геометрия и принцип работы

## Замеры (08.06.2026)

### Платформа
- Между замками: **16 см**
- Между флажками концевиков: **7 см**
- Флажки смещены **4 см внутрь** относительно замков (к центру платформы)

### Каретка
- Между концевиками: **34 см**

### Полочка
- Между прорезями: **32 см**

## Схема (вид сверху)

    Платформа:     [замок]----16см----[замок]
                      |  4см            |  4см
                   [флажок]---7см---[флажок]

    Каретка:    [FRONT]--------34см--------[BACK]
                концевик                  концевик
                GPIO 7                    GPIO 20

    Полочка:    [прорезь]------32см------[прорезь]

## Принцип позиционирования

**Ключевое:** Флажки смещены внутрь специально — чтобы при срабатывании
концевика соответствующий замок уже был точно под прорезью полочки.

| Направление | Концевик | Флажок | Замок под прорезью |
|-------------|----------|--------|-------------------|
| К FRONT     | GPIO 7   | Передний | Передний (GPIO 12) |
| К BACK      | GPIO 20  | Задний   | Задний (GPIO 13)   |

## Логика операций

### Извлечение из заднего ряда (extract_rear)

    1. Tray -> BACK         # Задний флажок срабатывает, задний замок под полочкой
    2. REAR lock GRAB       # Захватили полочку задним замком
    3. Tray -> FRONT        # Везём полочку, задний замок держит
    4. REAR lock RELEASE    # Отпускаем задний
    5. Tray -> BACK         # Подводим передний замок под прорезь
    6. FRONT lock GRAB      # Перехватываем передним
    7. Tray -> FRONT        # Выносим полочку

**Результат:** полочка на каретке, **передний** замок держит

### Возврат в задний ряд (return_rear)

    1. Tray -> BACK         # Передний замок в позиции отпускания
    2. FRONT lock RELEASE   # Отпускаем передний
    3. Tray -> FRONT        # Подводим задний замок под прорезь
    4. REAR lock GRAB       # Перехватываем задним
    5. Tray -> BACK         # Везём полочку в заднюю стойку
    6. REAR lock RELEASE    # Полочка в ячейке
    7. Tray -> CENTER       # Возврат в исходную позицию

### Извлечение из переднего ряда (extract_front)

    1. Tray -> FRONT        # Передний флажок срабатывает, передний замок под полочкой
    2. FRONT lock GRAB      # Захватили полочку передним замком
    3. Tray -> BACK         # Везём полочку, передний замок держит
    4. FRONT lock RELEASE   # Отпускаем передний
    5. Tray -> FRONT        # Подводим задний замок под прорезь
    6. REAR lock GRAB       # Перехватываем задним
    7. Tray -> BACK         # Выносим полочку

**Результат:** полочка на каретке, **задний** замок держит

### Возврат в передний ряд (return_front)

    1. Tray -> FRONT        # Задний замок в позиции отпускания
    2. REAR lock RELEASE    # Отпускаем задний
    3. Tray -> BACK         # Подводим передний замок под прорезь
    4. FRONT lock GRAB      # Перехватываем передним
    5. Tray -> FRONT        # Везём полочку в переднюю стойку
    6. FRONT lock RELEASE   # Полочка в ячейке
    7. Tray -> CENTER       # Возврат в исходную позицию

## Почему нужен перехват?

Полочка широкая (32 см между прорезями), платформа узкая (16 см между замками).

- При извлечении из **задней** стойки: захватываем **задним** замком,
  но чтобы вывезти полочку вперёд — перехватываем на **передний**
  
- При извлечении из **передней** стойки: захватываем **передним** замком,
  но чтобы вывезти полочку назад — перехватываем на **задний**

Это позволяет использовать один и тот же код для обеих стоек,
меняя только направления и замки.

## Константы в коде

| Параметр | Значение | Описание |
|----------|----------|----------|
| LOCK_DISTANCE | 12600 шагов | Расстояние между точками перехвата |
| TRAY_CENTER | 11300 шагов | Центральная позиция |
| TRAY_FREQ | 12000 Гц | Частота шагов (плавность) |

## GPIO распиновка

| Функция | GPIO | Примечание |
|---------|------|------------|
| TRAY_STEP | 18 | Шаги мотора платформы |
| TRAY_DIR | 27 | Направление (0=FRONT, 1=BACK) |
| TRAY_EN1 | 25 | Enable мотора 1 |
| TRAY_EN2 | 26 | Enable мотора 2 |
| ENDSTOP_FRONT | 7 | Передний концевик (1=нажат) |
| ENDSTOP_BACK | 20 | Задний концевик (1=нажат) |
| LOCK_FRONT | 12 | Передний замок (PWM сервопривод) |
| LOCK_REAR | 13 | Задний замок (PWM сервопривод) |
