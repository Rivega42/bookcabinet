# BookCabinet — Интеграция с ИРБИС64

## Подключение

> ВНИМАНИЕ: боевые адрес/логин/пароль ИРБИС в репозитории НЕ хранятся.
> Задаются только через переменные окружения (см. `.env.example`,
> `config.py` → `IRBIS`). Ниже — плейсхолдеры.

### Сервер
- **IP:** `<IRBIS_HOST>` (env `IRBIS_HOST`)
- **Port:** `<IRBIS_PORT>` (env `IRBIS_PORT`, по умолчанию 6666)
- **Login:** `<IRBIS_USERNAME>` (env `IRBIS_USERNAME`)
- **Password:** `<IRBIS_PASSWORD>` (env `IRBIS_PASSWORD`)

### Базы данных
- **KAT%SERV09%** — каталог книг
- **RDR** — база читателей

## Поиск

### Книги по RFID
Индекс: **`IN=`** с полным RFID

```
"IN=<RFID_МЕТКИ_ЭКЗЕМПЛЯРА>"
```

**Поле с RFID:** 910^H или 941^H

### Читатели по карте
Индексы: **`RI=`**, **`RFID=`**, **`CCUID=`**

```
"RI=<UID_КАРТЫ_ЧИТАТЕЛЯ>"
```

## Протокол ИРБИС64

### Формат пакета
```
[длина]\n[данные]
```

### Команды
| Код | Описание |
|-----|----------|
| A | Регистрация |
| B | Разрегистрация |
| C | Чтение записи |
| D | Сохранение записи |
| K | Поиск |

### Пример регистрации (Python)
```python
params = ('A', 'C', 'A', proc_id, cmd_num, '', '', '', '', '', login, password)
packet = '\n'.join(params)
packet = str(len(packet)) + '\n' + packet
```

### Пример поиска
```python
params = ('K', 'C', 'K', proc_id, cmd_num, '', '', '', '', '',
          db_name,      # KAT%SERV09%
          expr,         # "IN=RFID"
          '10',         # кол-во записей
          '1',          # первая запись
          '@brief',     # формат
          '', '', '')
```

### Формат ответа поиска
```
K\r\n
[server_id]\r\n
[cmd_num]\r\n
[length]\r\n
...\r\n
[status]\r\n      <- строка 10
[count]\r\n       <- строка 11, количество найденных
[mfn]#[brief]\r\n <- строка 12+, результаты
```

## Структура записи книги (MARC)

| Поле | Подполе | Описание |
|------|---------|----------|
| 200 | ^A | Заглавие |
| 700 | ^A | Автор (фамилия) |
| 700 | ^B | Автор (инициалы) |
| 210 | ^C | Издательство |
| 210 | ^D | Год издания |
| 910 | ^H | **RFID метка экземпляра** |
| 910 | ^B | Инвентарный номер |
| 910 | ^D | Место хранения (филиал) |
| 910 | ^A | Статус (0=доступна, 1=выдана) |
| 941 | ^H | RFID (дубль для индексации) |

## Проверено 14.04.2026

Связка «RFID метки экземпляра → каталожная запись» и «UID карты → читатель»
подтверждена на живом стенде. Конкретные RFID/UID и ПДн читателей в репозитории
не приводятся (152-ФЗ) — сверяются напрямую в ИРБИС.

| RFID | Книга |
|------|-------|
| `<RFID_1>` | `<книга 1>` |
| `<RFID_2>` | `<книга 2>` |

| Карта | Читатель |
|-------|----------|
| `<UID_КАРТЫ>` (NFC) | `<читатель, билет>` |
